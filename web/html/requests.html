Package requests contains two interfaces GetRequest and PostRequest

com.server.requests.GetRequest

Describes request method get

Map<String, String> execute(Map<String, String> params);

Gets get parameters and returns user's variables.

com.server.requests.PostRequest

Describes request method post

Map<String, String> execute(Map<String, String> getParams, Map<String, String> postParams);

Gets get and post parameters, returns user's variables.
