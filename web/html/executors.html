Package executors contains two classes: Connection and View

com.server.executors.Connection

Class Connection is a Thread. It gets Socket and handles connection with browser

Connection(Socket socket)

Constructor gets he socket to handle

void run()

Overrided method of superclass Thread. Gets InputStream and OutputStream from socket, makes requests and responses.

com.server.executors.View

It is util class. It forms html page using get and post requests, user's variables, maps css, js and other util files.

View(PageMapperBuilder.PageMapper pageMapper, OutputStream outputStream, Map<String, String> request)

Constructor gets all necessary entities to form html page and send it to request

void postExecute(PostRequest postRequest, Map<String, String> getParams, Map<String, String> postParams)

Executes post response using get and post parameters

getExecute(GetRequest getRequest, Map<String, String> getParams)

Executes get response using get parameters
