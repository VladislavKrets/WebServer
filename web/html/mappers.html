Package mappers contains two classes: PageMapperBuilder and PageMapperContainer

com.server.mappers.PageMapperBuilder

This class realizes Builder pattern using internal class PageMapper.
These two classes forming html page mapping procedure.

PageMapperBuilder()

Constructor without parameters, creates new PageMapper object.

PageMapperBuilder setPath(String path)

Sets url path for page. For example /server

PageMapperBuilder setPageName(String pageName)

Sets relative path from project to html file.

PageMapperBuilder setGetRequest(GetRequest getRequest)

Sets get request as lambda expression. Params are get parameters from request.
Get request always returns Map of variables.
You can create your variables to put them from java code to html file.
You can see syntax from this example:

... .setGetRequest(params -> { Map<String, String> vars = new HashMap<>(); vars.put("hello", "Hello World"); return vars; } ...

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> </head> <body> ${hello} </body> </html>

When you are importing different files like images, js files, etc by attribute src, or like css files by attribute href,
you should duplicate path to them, using attribute ${src}

<link rel="stylesheet" type="text/css" href="../css/mappers.css" ${src}="../css/mappers.css">

or

<img src="example.jpg" ${src}="example.jpg">

else you should map them by hands.

PageMapperBuilder setPostRequest(PostRequest postRequest)

Has the same rules as get request. Gets post and get parameters as arguments.

PageMapperBuilder setHtml(String html)

Gives an opportunity to set html from code without using files.

PageMapper getPageMapper()

Returns builded PageMapper entity

com.server.mappers.PageMapperContainer

It is a container for all mapped pages. Realizes pattern singleton.

PageMapperContainer getInstance()

Returns an instance of PageMapperContainer

synchronized void addPage(PageMapperBuilder.PageMapper pageMapper)

Adds PageMapper to the container

PageMapperBuilder.PageMapper getPage(String path)

Gets PageMapper from container by path

void deletePage(String path)

Deletes PageMapper from container by path
